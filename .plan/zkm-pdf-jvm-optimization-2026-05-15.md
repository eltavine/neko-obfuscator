## Scope

Use `docs/ZKM.pdf` as architecture input to improve the current JVM-only obfuscation pipeline, with priority on runtime performance and obfuscation strength. - Fresh baseline artifact directory is `build/test-jvm-full-obf-perf/`; it currently contains four JVM full-obf jars: `TEST-full-jvm-obf.jar`, `obfusjack-full-jvm-obf.jar`, `SnakeGame-full-jvm-obf.jar`, and `evaluator-full-jvm-obf.jar`.
